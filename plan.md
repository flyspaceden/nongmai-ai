## 1. 目标与定位
### 1.1 核心定位（CN）
以**5000亩生态鱼塘**为核心载体，打造“**水产养殖为基、科技体验为魂、文旅休闲为翼**”的复合型农文旅标杆，构建“可看、可玩、可学、可住、可购”的全链条体验，形成“农业 + 科技 + 文旅 + 研学”四位一体模式，成为广佛肇、珠中江城市群短途度假首选地。

### 1.2 Core Positioning (EN)
Built around a **5,000-mu eco fishpond landscape**, this destination combines **aquaculture**, **AI-powered science experiences**, and **culture & leisure tourism** to deliver an end-to-end journey: **see, play, learn, stay, and shop**—a distinctive “Agriculture + AI + Tourism + Education” model for short getaways across the Greater Bay Area.

### 1.3 目标用户与转化目标（按优先级）
1) **预约参观**（家庭/年轻客群/周边游客）  
2) **研学合作**（学校/教育机构/营地）  
3) **团建会务**（企业/协会/论坛活动）  
4) **住宿餐饮**（民宿、餐饮、烧烤、露营）  
5) **采购合作**（水产品/农产品/品牌联名与渠道）

## 3. 设计风格（AI科技大厂感）
### 3.1 视觉关键词
“**深海蓝 + 生态绿 + 科技青**”、数据可视化、玻璃拟态（Glassmorphism）、大留白、模块化卡片、渐变光晕、细腻动效。

### 3.2 色彩建议（可作为 Design Tokens）
- Primary（科技深蓝）：`#0B1220`  
- Secondary（海水青）：`#14B8A6`  
- Accent（生态绿）：`#22C55E`  
- Highlight（晨曦金，用于节庆/CTA点缀）：`#F59E0B`  
- Surface（卡片底）：`rgba(255,255,255,0.06)`（暗色主题）  
- Border：`rgba(255,255,255,0.12)`

### 3.3 字体与排版
- 中文：Noto Sans SC / Source Han Sans  
- 英文：Inter  
- 字级体系：Hero(48–64) / H2(32–40) / H3(20–24) / Body(16–18)  
- 栅格：12列，Max width 1200–1280px，左右边距 24px

### 3.4 组件与动效（静态但“像大厂”）
- 顶部导航：透明/毛玻璃，滚动后变实底；支持 Mega Menu  
- Hero：视频/大图背景 + 渐变遮罩 + 关键卖点标签（Pill）  
- 数据区：3–5个“关键数字”卡片（轻量计数动效可选）  
- 场景卡片：图标 + 标题 + 3条要点 + CTA  
- 时间轴/流程：研学课程、预约流程、追溯流程  
- “AI模块”统一样式：深色面板 + 图表/波形/SVG线框 + 解释文案
