Master Plan

五大分区，一张AI驱动的生态体验图

从渔耕体验到科技研学，从亲子互动到度假配套，形成分层动线与全龄体验。

生态农业体验区

鱼-菜-花-药循环，采摘认养、科普标识、四季可体验。

科技研学核心区

智慧实验室、农机文化博物馆、系统化研学课程。

亲子游乐互动区

分龄摸鱼垂钓、自然无动力游乐、亲子手作。

休闲度假配套区

主题民宿、生态餐饮、露营与夜间活动。

节庆赛事活动区

四季主题节庆 + 团建研学定制服务。

亲子一日游

上午渔耕种植 → 午间亲子厨房 → 下午摸鱼/手作体验。

研学一日营

实验室课程 + 农机互动 + 生态循环课程 + 结营证书。

两日轻度假

白天采摘+研学，夜晚露营星空；次日海陵岛联动。
